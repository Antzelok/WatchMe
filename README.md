# WatchMe

[Live Demo](watchme-production.up.railway.app)

WatchMe is a **premium e-commerce web application** showcasing high-quality wristwatches with precision, style, and lasting quality. Built with **Next.js** and **TypeScript**, it offers a fast, modern, and fully responsive user experience across devices.

## Features
- Modern product showcase layout  
- Responsive design for desktop and mobile  
- Fast performance with Next.js optimizations  
- Clean, component-based code structure  
- Easy deployment on Railway  

## Tech Stack
- **Next.js** – React framework for SSR and static site generation  
- **React** – UI library  
- **TypeScript** – Static typing for maintainable code  
- **CSS / Tailwind CSS** – Styling  
- **Railway** – Hosting and deployment  

## Getting Started
### Prerequisites
- Node.js v18+  
- Package manager: npm, yarn, or pnpm  

### Installation
```bash
git clone https://github.com/Antzelok/WatchMe.git
cd WatchMe
npm install   # or yarn install / pnpm install
npm run dev   # or yarn dev / pnpm dev
